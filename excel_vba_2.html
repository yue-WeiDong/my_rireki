excel数据筛选.txt
Excel商品数据处理的VBA宏集合，主要功能如下：
数据清洗：删除无关列、过滤无效数据（价格为0、下架商品）
价格管理：从外部表格提取活动价和成本价（如"活动&成本"、"限时设置"等工作表），计算最终售价
分类筛选：按商品类别（母婴、美妆、个护等）自动筛选并导出独立文件
网页代码生成：为商品生成HTML代码片段，可直接用于网站展示
统计功能：按一级/二级分类统计商品数量
格式优化：统一行高、列宽、字体和颜色标记
文件管理：自动创建带时间戳的备份文件，避免覆盖原始数据

Function IsFileExists(ByVal strFileName As String) As Boolean
    Dim objFileSystem As Object
    Set objFileSystem = CreateObject("Scripting.FileSystemObject")
    If objFileSystem.FileExists(strFileName) = True Then
        IsFileExists = True
    Else
        IsFileExists = False
    End If
End Function

Sub 分类统计()

End Sub

Sub 加入网址()
    Columns("M:M").Insert Shift:=xlShiftToRight
    Range("M1") = "网址"
    maxRow = Application.CountA(Range("a:a"))
    For i = 2 To maxRow
        Cells(i, "M") = "=CONCATENATE(""http://www.kjt.com/product/detail/"",B" & i & ")"
    Next i
End Sub
